## Community Programs

### Good First Issues
Labeled issues in every repository designed for new contributors. Each includes:
- Clear description of the problem
- Code pointers
- Testing instructions
- Mentor contact

### Contributor Ladder
1. First-time contributor (1 PR merged)
2. Repeat contributor (5 PRs merged)
3. Active contributor (10 PRs or 3 months)
4. Community maintainer (invited, sustained contribution)
5. Core maintainer (employee or elected)

### Recognition
- Contributors listed in repository CONTRIBUTORS file
- Quarterly contributor spotlight on blog
- Annual contributor survey and feedback session
- Signed thank-you postcards for significant contributions
